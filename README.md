# HealthInfo Website

A modern health information platform built with React.js. It features user authentication, video uploads, and a responsive design for all devices. The site also supports audio translations (English and Bangla) and article narration for accessibility.

## Project Status

This project is fully developed and production-ready.

## Features

- **Modern UI**: Clean, health-focused design
- **Authentication**: Login/logout with protected routes
- **Video Upload**: Upload and display videos
- **Responsive Design**: Optimized for all devices
- **Audio Translation**: Supports English and Bangla
- **Article Narration**: Translate and read articles aloud
